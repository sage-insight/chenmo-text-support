# Sage Insight - Text-Only Digital Support Landing Pages

Public pages in this folder are bilingual (Chinese / English) and automatically switch based on browser language and timezone.

## Public files only
- `index.html` - main landing page
- `start-here.html` - quick orientation page
- `payment.html` - payment and order info
- `order-now.html` - direct order page
- `i18n.js` - language detection and switching
- `usdt-trc20-qr.png` - payment QR code

## Security rule
Do not place project planning documents, private operational notes, deployment checklists, or security-sensitive files in this public folder.
Sensitive docs must stay outside `deploy/`.
